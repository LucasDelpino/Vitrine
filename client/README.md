# Vitrine - Application e-commerce

Application e-commerce en cours de refonte avec :

- Front : React + Vite
- Back : Node.js + Express
- Base de données : MySQL
- Paiement : Stripe

## Structure du projet

- `client/` : interface utilisateur React
- `server/` : API Express
- `db_init/` : scripts SQL d'initialisation / migration
