# Vitrine

Projet e-commerce avec React, Vite, Node.js, Express, MySQL, Docker et Stripe.

## Structure

- `client/` : frontend
- `server/` : backend
- `db_init/` : scripts SQL
- `docker-compose.yml` : MySQL + phpMyAdmin
